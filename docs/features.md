# VoltMaster Frontend Features

- Customer-facing service catalog for residential, commercial, EV, solar, backup power, emergency repair, and lighting.
- Instant quote estimator with project type, size, timeline, budget, notes, and photo upload fields.
- Filterable project portfolio with sample commercial, residential, EV, and renewable projects.
- Booking board for consultation windows, site visits, inspection corrections, and emergency dispatch.
- Credentials page for license, insurance, training, and years-in-business trust signals.
- Contact and resource pages for lead capture and safety-focused content.
